- Red: #ef4444 (Errors, Warnings)
- Yellow: #eab308 (Pending, Info)
- Gray: #6b7280 (Text, Borders)

- White: #ffffff (Content)
- Light Gray: #f9fafb (Page background)
- Lighter Gray: #f3f4f6 (Hover states)

## Responsive Behavior

```
Desktop (1024px+):
- Sidebar navigation on left
- Content on right
- 2-column layout
- Full width forms

Tablet (640px - 1024px):
- Sidebar navigation on left
- Content on right
- Narrower layout
- Adjusted spacing

Mobile (< 640px):
- Stacked navigation
- Full width content
- Single column
- Touch-friendly buttons
- Larger tap targets
```

## Animations

```
Transitions:
- Hover effects: 200ms ease
- Tab switching: 150ms ease
- Loading spinner: Continuous
- Success message: Fade in/out

Loading States:
- Skeleton screens
- Spinner animation
- Disabled buttons
- Opacity changes

Error States:
- Red border on input
- Error message below
- Icon indicator
- Shake animation (optional)

Success States:
- Green checkmark
- Success message
- Fade out after 3s
- Smooth transition
```

```

- Arrow keys for navigation

- Form labels
- ARIA descriptions
- Status announcements
- Error messages

Color Contrast:
- WCAG AA compliant
- 4.5:1 minimum ratio
- Clear visual hierarchy
- No color-only indicators
```
